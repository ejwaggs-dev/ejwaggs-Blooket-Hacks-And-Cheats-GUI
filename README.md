⚙️ Setup & Usage Instructions
1. Using Bookmarklets

Bookmarklets are buttons basically in Bookmarks Bar when clicked can activate the code embedded inside of it, executing and showing the GUI on the website.

    Go to the Cheats folder in the main directory.
    Pick the file that includes min.js towards the end
    Copy the code altogether
    Create a bookmark and locate it on the Bookmarks Bar
    Make sure to edit it afterwards with an optional name like "Blooket Cheats", but paste the code that you copied into the URL area, make sure the code starts with "Javascript:"
    Click the bookmark while on the Blooket game page to activate the cheat.

2. Using the Browser Console

You can also run scripts directly in the browser console:

    Go to the Cheats folder in the main directory.
    Pick the file that includes .js towards the end
    Copy the code altogether
    Press F12 or Ctrl+Shift+I to open developer tools.
    Go to the "Console" tab.
    Paste the code into there and press Enter to activate the GUI
